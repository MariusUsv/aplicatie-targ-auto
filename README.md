# Aplicatie practica – Gestionare tranzactii targ auto

## Tema
Aplicatia are ca scop gestionarea tranzactiilor de vanzare–cumparare realizate intr-un targ auto.

## Date stocate pentru fiecare tranzactie

Pentru fiecare masina tranzactionata se vor inregistra:

- Nume vanzator
- Nume cumparator
- Firma masinii
- Modelul masinii
- An fabricatie
- Culoare
- Optiuni
- Data tranzactie
- Pret

## Functionalitati

Aplicatia permite:

- introducere tranzactie
- editare tranzactie
- stergere tranzactie

## Rapoarte

Aplicatia va afisa:

- cea mai cautata masina intr-o anumita perioada
- graficul pretului pentru un model
- tranzactiile dintr-o anumita zi

## Validari

La introducerea unei tranzactii:
- daca cineva cumpara mai multe masini intr-o zi apare avertizare
- daca cineva vinde mai multe masini intr-o zi apare avertizare
